# System Health Checker — Bash Script


A simple Bash automation script that displays key system information including:
- Uptime
- Logged-in users
- CPU usage
- Memory usage
- Disk usage
- Running processes

This project helps me practice Linux administration, Bash scripting, and system monitoring concepts.

---

## Skills Demonstrated
- Linux CLI
- Bash scripting
- File permissions
- System resource monitoring
- Automation fundamentals

---

## How to Run

chmod +x system_health.sh
./system_health.sh
